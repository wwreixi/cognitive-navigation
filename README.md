# 🧭 Cognitive Navigation

**Runtime Safety Diagnosis for AI Agents**

[![Python](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)

Cognitive Navigation is a diagnostic framework that monitors AI Agent health in real-time using the **S = T - C - D** model — a mathematical formulation grounded in the Yiyongxue theory of transformative-constraint dynamics.

---

## 🎯 Why Cognitive Navigation?

AI Agents are increasingly deployed in critical workflows — yet their behavior remains opaque and unpredictable. Cognitive Navigation provides:

| Capability | What it does |
|------------|--------------|
| **Real-time Health Scoring** | Computes S (Health) from T (Transformative Force), C (Constraint Force), and D (Internal Dissipation) |
| **Four-Zone Alert System** | GREEN / YELLOW / ORANGE / RED with actionable navigation instructions |
| **Context Compression (V4.0)** | Reduces token consumption by up to 62.1% while preserving critical information |
| **Multi-Agent Consistency** | Ensures all agents in a collaborative workflow share the same health assessment |

## 🚀 Quick Start

```bash
# 1. Run the Cognitive Navigation API via Docker
docker run -d -p 8000:8000 cognitive-nav:latest

# 2. Call the diagnosis endpoint
curl -X POST http://localhost:8000/diagnosis \
  -H "Content-Type: application/json" \
  -d '{"query": "Your agent\'s output or task description"}'

# 3. Receive S/T/C/D/zone/nav_instruction
```
